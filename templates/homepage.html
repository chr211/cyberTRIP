Welcome to CyberTRIP: Cyber Triage & Response Incident Platform

A comprehensive web-based tool designed to track, manage, and prioritize cyber security incidents in real-time. Integrated with leading external cybersecurity APIs, CyberTRIP provides an all-in-one solution to swiftly respond to threats, ensuring digital assets remain secure. The application will provide functionality to add and save incidents by name, type, location, and more. It will be able to store them in a searchable database to be able to search and generate statistics. All data can be exported to csv or similar formats. Tools associated with different types of incident response will be accessible on the dashboard via API.
